Візуалізація зв’язків публічної особи Visualization of connections of politically exposed person

Зв'язки з компаніями, родинні, особисті, політичні та інші зв’язки відображаються за допомогою візуалізації. Ви можете дізнатися, чи публічна особа є власником або бенефіціаром, чи має офіційну посаду в управлінні компанії. Connection to companies, family, personal, political and other connections are displayed through a visualization. You can find out if a public person is the owner or the beneficiary, or has an official position in the management of the company.

Кому потрібен реєстр Who needs a register

Фінансова установа зобов'язана виявити публічних діячів серед своїх клієнтів та встановити джерело їх статків. За допомогою реєстру банки, страхові компанії, платіжні системи, аудитори, установи фінансового лізингу вже перевіряють клієнтів. A financial institution is obliged to identify politically exposed persons among its clients and establish the source of their financial condition. With the help of the register, banks, insurance companies, payment systems, auditors, and financial leasing institutions are already checking customers.

Доступ до реєстру через REST API An access to the registry, using the REST API

Надаємо доступ до реєстру через REST API для зручної інтеграції у Ваші корпоративні системи та інших потреб. Отримати дані можна у форматах XML та JSON. Access to the registry is given through the REST API for easy integration into your corporate systems and other needs. Getting the data is possible in XML and JSON formats.
